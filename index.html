中国合伙人

发表于 2016-04-12   |   分类于 随笔   |     |   阅读次数
  • 梦想是什么，梦想就是你坚持觉得是幸福的东西。
  • 昨天晚上找个时间把中国合伙人又看了一遍，记得第一次看中国合伙人是在高三的时候，那个时候班级里刚安投影没多久，班级里同学商量生物老师（一位年轻的长得很壮的男性，2333。。。）带我看一场电影，生物老师就给找了一个这个电影。一转眼也过去这么久了，自己也即将面向就业，在工作稳定以后要找个时间回去看看高中的老师，想一想曾经那最美好的岁月，那个曾经最喜欢的姑娘。
  • 这个电影真的是百看不厌，当然电影里的新梦想就是今天的新东方，影片记录了俞敏洪与他的两位好友的创业史以及他们炙热的友情。看过电影以后自己又查了一下柴静看见对三位创始人的采访。三位boss也是给了最真诚的回答。用俞敏洪的话来讲，他的那些北大的同学当初愿意与他有来往并不是因为他有才华而是他人好，这可以看出他在上大学时期其实是自卑的，在他的同学中他最钦佩的也是他当时的班长王强。在电影中主角在办理签证时也有体现，签证官问他最佩服谁，他说孟晓俊。
  • 王强这个人我比较喜欢，说话不藏着掖着。他从美国回来以后，帮助俞敏洪做了很多事情，特别去家族化这个事，我也认为是特别正确的。王强为新东方制定了很多管理规则，比如不允许在教学区吸烟，这个规则刚实施的时候，第一个处分的就是俞敏洪的母亲，新东方的去家族化由此开始，事实证明这个决定是非常正确，真正想要做大一个企业，一定不能有家族的存在。当主持人柴静问王强你有没有想过你这样做没有给俞敏洪留面子的时候，王强说，我要是那样纵容我就不是王强了，这句话对我感触挺深的。
  • 现在新东方已经是一家上市公司，在采访中还是可以看出来俞敏洪有点后悔新东方上市的，用他自己的话说就是资本市场就像你娶了一个老婆你甩又甩不掉，你还要一年保持你企业的百分之二十的利润，所以这就导致了新东方的急速扩张，做教育这一块一定是不能讲速度的。
  • 未完待续。。。以后想到哪些再来补充。

百度卫士8.0发布会后的一点想法

发表于 2016-04-12   |   分类于 产品   |     |   阅读次数
  • 前几天同学有一个参加百度卫士8.0发布会的机会，他因为时间问题去不了了，我就去参加了一下。因为毕业以后想往产品这方面发展，所以去听了一下，收货还是挺多的。
  • 刚开始大boss技术总监说了一下百度卫士的一些优秀成果，技术部门配合警方打掉了哪些哪些黑色窝点，颇为震惊的是他说了一个曾经与北京警方合作打掉的制造“网络病毒”的窝点，这还是冰山一角，打掉的只是负责制造的，而负责传播的等等还有很多。当然，制造病毒的那些人在技术层面上肯定要高于现阶段的我，而他们并没有学有所用，这也给即将走向工作岗位的我有了一个警示作用，以后要做事有原则，做人有规矩。
  • 印象最深的还是一个设计负责人的分享，应该也是在二十七八作用。他说百度卫士目前的主要用户人群集中在20岁在35岁之间，所以设计的时候站在这个年龄段的人的角度思考的比较多。百度卫士8.0版本也是继续秉承简洁的原则，让用户最最少的判断，比如一个抽屉效果的实现，这就像现实中的抽屉一样，每个人在抽屉关着的时候就会本能的去拉，而当一个门关着的时候，人就要判断我是否推门还是拉门。而对于百度卫士的主色调，是一个深蓝色的主色调，他解释的原因是，百度卫士是做安全软件的，首先要考虑的是安全的问题，所以在颜色设计上选择一个偏男性的色调，有一种安全感。这个主色调在不同的软件上是不一样的，比如微博的有点类似与橘黄色，而微信的是白色和绿色，给人一种简洁的感觉。说到微信，还记得好几年以前，那个时候用的是塞班系统的诺基亚触屏手机，那个是我人生中第一步智能手机，有一个女同学在QQ上跟我说，腾讯又有一款新产品了叫微信，咱俩赶快注册，能注册到比较好的账号，应该距离现在没有几年吧，现在微信已经是一款很成功的产品了。
  • 最后那个设计负责人又讲了一下，人在用手机软件的时候，百分之八十的时间会用一些常用的功能，而百分之二十的时间会用在不常用的功能，所以百度手机卫士把一些不常用的功能模块都放在了“百宝箱”里，一提到百宝箱，大家会很容易想到我会在里面找到我很多想要的东西。
  • 以后还是应该多多参加一些这样的活动，可以在其中学习到很多东西，包括他们在台上的说话方式，与别人分享的方式都很值得去学习。
